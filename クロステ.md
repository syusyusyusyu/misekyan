# Cross Stage（クロステ） セットアップ & 遊び方ガイド

> 歌詞を「つかんで」「奏でる」 新感覚リズムゲーム

**URL:** https://cross-stage.c2375.workers.dev/

---

## はじめに

Cross Stage は、画面に流れてくる **歌詞バブル** を掴んでホールドするリズムゲームです。
マウス・スマホタッチだけでなく、カメラを使って **全身** や **顔** でも操作できます。

楽曲は **「ストリートライト」（ネギシャワーP）** です。

---

## 必要なもの

| モード | 必要なもの |
|:---|:---|
| Cursor（PC） | PC + マウス |
| Mobile（スマホ） | スマートフォン |
| Body（全身） | PC + Webカメラ（全身が映る距離） |
| Face（顔） | スマホ + インカメラ |

- ブラウザは **Chrome** 推奨

---

## 遊び方（3ステップ）

### ステップ 1 ： モードを選ぶ

サイトにアクセスすると、タイトル画面が表示されます。
左側のタブから操作モードを選んでください。

| モード | こんな人向け |
|:---|:---|
| **カーソルモード** | PCでサクッと遊びたい |
| **モバイルモード** | スマホで手軽に遊びたい |
| **ボディモード** | 全身を使って踊りたい！ |
| **フェイスモード** | 顔の動き＋口パクで遊びたい |

### ステップ 2 ： 速度コースを選ぶ

バブルが画面に表示される時間を選びます。

| コース | 表示時間 | 難易度 |
|:---:|:---:|:---:|
| 12秒 | ゆっくり | やさしい |
| 10秒 | ふつう | おすすめ |
| 8秒 | はやい | むずかしい |

初めての方は **12秒コース** がおすすめです。

### ステップ 3 ： ゲームスタート！

「**ゲームスタート**」ボタンを押して、プレイ開始です。

## 操作方法（モード別）

### カーソル モード（PC）

1. 再生ボタンを押すと楽曲スタート
2. 画面下から **歌詞バブル** が浮かんでくる
3. バブルにマウスカーソルを合わせて **長押し（ホールド）**
4. 円形ゲージが 100% になるとコンボ継続！

### モバイル モード（スマホ）

1. 再生ボタンをタップして楽曲スタート
2. 浮かんでくるバブルを **指でタッチ＆長押し**
3. ゲージを溜めてスコアを稼ごう

### ボディ モード（全身カメラ）

1. カメラへのアクセスを許可する
2. **全身がカメラに映るように** プロジェクターから離れて立ち位置を調整
3. 全身が認識されると **5秒カウントダウン** → 自動で再生開始
4. **手や腕、足をバブルに重ねて** ホールド判定
5. 全身がカメラから外れると警告が出るので注意

### フェイス モード（顔カメラ）

1. カメラへのアクセスを許可する
2. 顔がカメラに映ると自動でスタート
3. **鼻先** がカーソル代わり → バブルに鼻を合わせる
4. **口を「あ」と大きく開ける** とホールド判定
5. 口を閉じるとホールド終了

## スコアとランク

### スコアの仕組み

- バブルをホールドしている間、スコアが **段階的に加算** される
- ゲージ 100% 達成で **コンボ** が繋がる
- 途中で離すとコンボリセット

### ランク

| ランク | 評価 |
|:---:|:---|
| **S** | パーフェクト！ |
| **A** | すごい！ |
| **B** | いい感じ！ |
| **C** | もう一回チャレンジ！ |

## スコア登録 & ランキング

1. 曲が終わるとリザルト画面が表示される
2. **プレイヤー名** を入力
3. Bot対策の認証（Turnstile）を通す
4. 「登録」でランキングに反映！

ランキングはタイトル画面の「**ランキング**」ボタンから確認できます。
期間（全期間 / 週間 / 24時間）やコース別に絞り込めます。

## よくある質問

**Q. 音が出ない**
→ ブラウザの音量設定とデバイスの音量を確認してください。

**Q. カメラが使えない**
→ ブラウザのカメラ許可設定を確認してください。アドレスバー横の鍵アイコンから許可できます。

**Q. Body モードで認識されない**
→ 全身（頭から足先まで）がカメラに映るように、カメラから離れてください。

**Q. バブルが速すぎる**
→ 12秒コース（やさしい）を試してみてください。

**Q. スマホでも遊べる？**
→ 遊べます！ フェイス モードがおすすめです。もう1台で用意しているスライドを表示してQRコードから読み取ってください

**Q. ホーム画面に追加できる？**
→ はい。PWA対応なので、ブラウザの「ホーム画面に追加」からアプリのように起動できます。

## 注意

・ボディモードを高校生の子にプレイしていただく際は、足元のコンセントが飛び出て転倒するリスクも説明したうえでプレイに前向きな子のみにやらせる。基本はSAがデモで高校生にはモバイルモードとフェイスモードを推奨。
→スマホでも遊べるための表示QRコードはパネル付近のPCのパワーポイントのスライドを表示してください。
・プロジェクターに映すPCは吉田がMF後放置するのでそれを使ってください。（学校のPCではドメインがブロックされるため。）
・PCのPINは一部のSAのみに教えます。
・ボディモードはPCのスペック上反応が良くないです。
